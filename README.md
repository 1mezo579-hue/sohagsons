# أبناء سوهاج - نظام الكاشير المتكامل (نسخة السوبر ماركت)

نظام إدارة سوبر ماركت أبناء سوهاج - نسخة احترافية تعمل أوفلاين مع مزامنة سحابية تلقائية.

## 🚀 كيفية التشغيل (للمرة الأولى)
1. تأكد من تثبيت **Node.js** على الجهاز.
2. قم بتشغيل ملف `تجهيز النظام.bat` وانتظر حتى ينتهي.
3. ابدأ العمل بالضغط على `ابدأ البرنامج.vbs`.

## 📂 ملفات التحكم الاحترافية
- **ابدأ البرنامج.vbs**: يفتح البرنامج في الخلفية وبشكل "تطبيق" صامت (بدون شاشات سوداء).
- **إغلاق البرنامج.vbs**: يغلق كافة عمليات النظام بضغطة واحدة.
- **عرض قاعدة البيانات.bat**: لوحة تحكم لتعديل البيانات يدوياً بسهولة.
- **اقرأني أولاً.txt**: دليل سريع لحل المشكلات.

## ✨ المميزات المتقدمة
- **العمل بدون إنترنت**: النظام يعتمد على قاعدة بيانات محلية (SQLite) لضمان السرعة القصوى وعدم التوقف.
- **المزامنة التلقائية**: خدمة خلفية تقوم برفع الفواتير إلى السحابة (Supabase) أوتوماتيكياً فور توفر الإنترنت.
- **واجهة مستخدم بريميم**: تصميم عصري وسهل التعامل مع دعم كامل للباركود والموازين.

## 🛠 الدعم الفني
في حال واجهت أي مشكلة، يرجى مراجعة ملف `TROUBLESHOOT.md` أو التواصل مع المطور.
