# أبناء سوهاج - نظام الكاشير

نظام إدارة سوبر ماركت أبناء سوهاج - كاشير ومخازن وتقارير

## المميزات

- كاشير سريع مع دعم الباركود والسكانر
- منتجات بالوزن (جبن، رز، مكرونات)
- منتجات بالقطعة (بهارات باكيت 10ج/15ج/20ج)
- توريد وصرف مخزني مع تتبع كامل
- تنبيهات المخزون المنخفض
- تقارير يومية وشهرية مع رسوم بيانية
- طباعة فواتير حرارية
- إدارة المستخدمين والصلاحيات
- واجهة عربية سهلة

## خطوات التشغيل

### 1. تثبيت الحزم

```bash
npm install --legacy-peer-deps
```

لو فشل:
```bash
# امسح المجلدات القديمة يدوياً ثم:
npm install --legacy-peer-deps
```

### 2. إعداد قاعدة البيانات

```bash
npx prisma db push
npx prisma generate
```

### 3. إضافة بيانات تجريبية

```bash
npx tsx prisma/seed.ts
```

### 4. تشغيل التطبيق

```bash
npm run dev
```

افتح المتصفح على: `http://localhost:3000`

## اختصارات الكاشير

| الزر | الوظيفة |
|------|---------|
| F2 | مسح الباركود |
| F3 | بحث بالاسم |
| F4 | إتمام البيع |

## الصفحات

| الصفحة | الرابط | الاستخدام |
|--------|--------|-----------|
| الرئيسية | `/` | الوصول للأقسام |
| الكاشير | `/cashier` | البيع اليومي |
| المخزن | `/inventory` | إضافة منتجات وتوريد |
| التقارير | `/reports` | مراجعة المبيعات |
| المستخدمين | `/users` | إدارة الصلاحيات |

## الرفع على Vercel

1. اعمل GitHub Repo جديد وارفع الملفات
2. ادخل [vercel.com](https://vercel.com) → New Project
3. اختار الـ Repo → Framework Preset: Next.js
4. Environment Variables:
   - `DATABASE_URL` = `file:./dev.db`
